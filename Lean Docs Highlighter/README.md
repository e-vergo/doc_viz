# Lean Docs Highlighter - Extension

See the [main README](../README.md) for full documentation.

## Quick Start

```bash
./build.sh
```

This builds the extension and opens a test page in Safari.

## Key Files

| File | Purpose |
|------|---------|
| `Shared (Extension)/Resources/content.css` | Dark theme + syntax highlighting |
| `Shared (Extension)/Resources/manifest.json` | Extension configuration |
| `build.sh` | Build and test script |
